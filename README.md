# Code in C for Ecomerce

Descrição curta do que o projeto faz.

## Funcionalidades

- Cadastro de clientes
- Cadastro de produtos
- Consultas
- Salvamento de dados em arquivos binários

## Como Usar

1. Compile o código com `gcc siagro.c -o siagro`
2. Execute com `./siagro`
3. Siga o menu para realizar as ações.
